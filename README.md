# Karamd – Job Recruitment Platform

## Description

Karamd is a professional job recruitment platform developed using Django.
The system was designed specifically for the computer and IT job market, allowing users to create, upload, and manage resumes while enabling employers to post job opportunities.

## Features

* Job posting system for employers
* Resume creation and submission for job seekers
* API-based architecture
* User role management
* Designed for IT and computer-related careers

## Tech Stack

* Python
* Django
* REST APIs
* Database integration

## Status

Development has been discontinued. The project remains available as a portfolio and reference implementation.
